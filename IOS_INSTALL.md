# Install on iPhone as a web app

The simple route:

1. Upload this folder to a static website host.
2. Open the site in Safari on iPhone.
3. Tap Share.
4. Tap Add to Home Screen.
5. Open TonyST from the home screen.

Good hosting options:

1. GitHub Pages
2. Netlify
3. Cloudflare Pages
4. Any HTTPS static hosting

Important:

Service worker caching requires HTTPS, except for localhost during development.
