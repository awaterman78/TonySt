Quick help

A is Fire in joystick mode, and left click in mouse mode. B and Start send the built in start sequence for awkward intros. Keyboard is there for Space, Enter, F keys and typing.

Use Load game for .st, .msa or .stx first. Screen mode toggles zoom if a game looks cropped. Input: Gamepad keeps joystick support steadier on iPhone.

Uses the existing Hatari engine files in engine/hatari and the Mesmotronic browser Hatari build as the working base.
