Quick help

Use Load game for .st or .msa first. Fire is joystick fire in joystick mode, and left mouse click in mouse mode. If a title screen wants Space, Enter or F1, use Keyboard or Send Start.

The Input button switches between virtual Gamepad and keyboard joystick mode, then reloads the emulator. Gamepad is the default because iPhone Safari is fussy with fake keyboard events.

Uses the Mesmotronic browser Hatari build and the existing engine files in engine/hatari.
