# TonyST Flat v0.5.2, engine/hatari patch

This patch changes TonyST so the emulator iframe loads from:

engine/hatari/tonyst.html

Upload these files to your GitHub Pages branch:

app.js
engine/hatari/tonyst.html

Your engine/hatari folder must also contain:

engine/hatari/hatari.data
engine/hatari/hatari.js
engine/hatari/hatari.wasm

Use lowercase `hatari`, not `Hatari`, to avoid path issues on GitHub Pages.
